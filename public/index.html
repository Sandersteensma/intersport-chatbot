Intersport Chatbot — Test pagina

Welkom! Dit is de testomgeving voor het Intersport chatbot widget. Klik rechtsonder op de rode bubble om te chatten.

⚙️ Admin dashboard: /admin/

Hoe plaats ik dit op mijn Magento 2 site?

Voeg in Magento Admin onder Content → Design → Configuration → HTML Head → Scripts and Style Sheets deze regel toe:

<script src="https://JOUW-SITE.netlify.app/widget/chatbot.js" async></script>

Features

  • ✅ Klantenservice / FAQ (instelbaar via admin panel)
  • ✅ Productadvies (live Magento zoekopdrachten)
  • ✅ Orderstatus opvragen op ordernummer
  • ✅ Track & trace codes opzoeken
  • ✅ Retourhulp
  • ✅ Feedback verzamelen (👍/👎)
  • ✅ Alle gesprekken opgeslagen — de bot leert via het admin panel
